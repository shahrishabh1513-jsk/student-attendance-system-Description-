# 📖 About The Project

The **Student Attendance System** is a front-end web application developed using **HTML, CSS, and JavaScript**.

It allows users to efficiently manage:

- 👨‍🎓 Student Information
- 📚 Subjects
- ✅ Attendance Records
- 📊 Attendance Reports

The project provides a clean and responsive interface with an organized folder structure for better maintainability.

---

# ✨ Features

## 👨‍🎓 Student Management

- Add Student
- Edit Student
- Delete Student
- View Student List

## 📚 Subject Management

- Add Subjects
- Update Subjects
- Delete Subjects

## ✅ Attendance

- Mark Attendance
- Store Attendance Records
- Easy Record Management

## 📊 Reports

- Attendance Summary
- Student-wise Reports
- Subject-wise Reports

## 🎨 User Interface

- Responsive Layout
- Clean Dashboard
- Modern Design
- Easy Navigation
